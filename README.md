# Eye-Filter
I made this project for fun using python libraries opencv, mediapipe, pygame, time , math, numpy. This is an eye-filter project , where ther system detects a user's eye blink and displays Itachi's eye (A character from Naruto anime) activation filter after the first blink.

Make sure you have installed these libraries before running the project:
pip install numpy pandas opencv-python mediapipe pygame
